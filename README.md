# TurismoPro (En Desarrollo)
Aplicación móvil desarrollada en Flutter para explorar destinos turísticos, visualizar información detallada de ciudades y municipios, y ofrecer una experiencia moderna, rápida y escalable.

---
[![GitHub Img](https://github.com/moleculax/TurismoPro/blob/main/assets/image/screen.png)](https://github.com/moleculax/TurismoPro/blob/main/assets/image/screen.png)

##  Características principales

-  **Exploración de ciudades y municipios**  
  Listado completo con imágenes, descripciones y datos relevantes.

-  **Mapa interactivo**  
  Visualización de ubicaciones y puntos de interés.

- **Galerías e imágenes online**  
  Integración con recursos remotos para mantener la app liviana.

- ⚡ **Arquitectura modular y escalable**  
  Código organizado en capas, widgets reutilizables y modelos tipados.

- 📱 **Compatibilidad multiplataforma**  
  Android, iOS, Web, Windows, Linux y macOS.

---




[![GitHub Img](https://github.com/moleculax/TurismoPro/blob/main/assets/image/screen3.png)](https://github.com/moleculax/TurismoPro/blob/main/assets/image/screen3.png)


## Tecnologías utilizadas

- **Flutter 3.x**
- **Dart**
- **Material Design 3**
- **HTTP / REST APIs**
- **JSON dinámico y modelos tipados**
- **Desarrollo ejecutando  Android Studio en GNU/Linux  (Debian)**
---

##  Estructura del proyecto

 ```bash
lib/
├── models/          # Modelos de datos tipados
├── screens/         # Pantallas principales
├── widgets/         # Componentes reutilizables
├── services/        # Conexión a APIs y lógica externa
├── utils/           # Helpers, constantes y funciones globales
└── main.dart         # Punto de entrada de la app
 ```



##  Instalación y ejecución

### 1. Clonar el repositorio

```bash
git clone git@github.com:moleculax/TurismoPro.git
cd TurismoPro
flutter run

```
